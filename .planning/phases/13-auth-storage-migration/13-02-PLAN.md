**src/services/supabase/storageService.ts:**

Import `supabase` from `@/lib/supabase`, `decode` from `base64-arraybuffer`, `* as FileSystem` from `expo-file-system/legacy`, `* as ImageManipulator` from `expo-image-manipulator`, `logger` from `../../utils/logger`.

Helper functions (not exported):
- `compressImage(uri: string, options: { width: number; quality: number; format?: 'webp' | 'jpeg' }): Promise<string>` -- Calls `ImageManipulator.manipulateAsync(uri, [{ resize: { width } }], { compress: quality, format: SaveFormat.WEBP or JPEG })`. Returns result.uri.
- `readFileAsArrayBuffer(uri: string): Promise<ArrayBuffer>` -- Reads file as base64 via `FileSystem.readAsStringAsync(uri, { encoding: FileSystem.EncodingType.Base64 })`, then returns `decode(base64)`.
- `getContentType(extension: string): string` -- Maps: `.webp` -> `'image/webp'`, `.jpeg`/`.jpg` -> `'image/jpeg'`, `.png` -> `'image/png'`, `.mov` -> `'video/quicktime'`, `.mp4` -> `'video/mp4'`. Default: `'application/octet-stream'`.

Exported functions -- all return `{ success: boolean; url?: string; storagePath?: string; error?: string }` pattern:

1. `uploadPhoto(userId, photoId, localUri)` -- Compress WebP 0.9 / 1080px -> read as ArrayBuffer -> `supabase.storage.from('photos').upload('photos/${userId}/${photoId}.webp', arrayBuffer, { contentType: 'image/webp', cacheControl: 'public, max-age=31536000', upsert: false })` -> `getPublicUrl` -> return `{ success: true, url, storagePath }`.

2. `uploadVideo(userId, photoId, videoUri)` -- Detect extension from URI (.mov or .mp4) -> read as ArrayBuffer (NO compression) -> upload to `photos/${userId}/${photoId}.{ext}` with correct contentType -> getPublicUrl -> return.

3. `uploadProfilePhoto(userId, localUri)` -- Compress WebP 0.7 / 400px -> upload to `profile-photos` bucket at path `${userId}/profile.webp` -> getPublicUrl from `profile-photos` bucket -> return.

4. `uploadSelectsPhotos(userId, localUris: string[])` -- Map over URIs with index, compress WebP 0.7 / 400px each, upload to `selects` bucket at `${userId}/select_${index}.webp` -> collect publicUrls -> return `{ success: true, photoURLs }`.

5. `uploadCommentImage(localUri)` -- Generate UUID for filename, compress WebP 0.9 / 1080px -> upload to `comment-images` bucket at `${uuid}.webp` -> getPublicUrl -> return.

6. `uploadSnapPhoto(userId, snapId, localUri)` -- Compress WebP 0.9 / 1080px -> upload to `snaps` bucket at `${userId}/${snapId}.webp` -> return `{ success: true, storagePath }` (NO publicUrl -- snaps are private).

7. `deletePhoto(userId, photoId)` -- `supabase.storage.from('photos').remove([`photos/${userId}/${photoId}.webp`])`. Try .webp first; if not found try .jpg fallback for pre-migration photos.

8. `deleteProfilePhoto(userId)` -- `supabase.storage.from('profile-photos').remove([`${userId}/profile.webp`])`. Same .jpg fallback.

9. `getPhotoURL(userId, photoId)` -- `supabase.storage.from('photos').getPublicUrl(`photos/${userId}/${photoId}.webp`)`. Returns `{ success: true, url }`.

10. `generateThumbnail(uri)` -- Resize to 20px wide JPEG 0.5 quality -> read as base64 -> return `data:image/jpeg;base64,${base64}` or null on error.

**supabase/functions/migrate-firebase-auth/index.ts:**

This Edge Function runs in Deno. Firebase Admin SDK does NOT work in Deno, so use the Firebase REST API to verify the ID token.

```typescript
import { createClient } from 'https://esm.sh/@supabase/supabase-js@2';

const corsHeaders = {
  'Access-Control-Allow-Origin': '*',
  'Access-Control-Allow-Headers': 'authorization, x-client-info, apikey, content-type',
};

Deno.serve(async (req) => {
  // Handle CORS preflight
  if (req.method === 'OPTIONS') {
    return new Response('ok', { headers: corsHeaders });
  }

  try {
    const { firebaseToken } = await req.json();
    if (!firebaseToken) {
      return new Response(JSON.stringify({ error: 'Missing firebaseToken' }), {
        status: 400,
        headers: { ...corsHeaders, 'Content-Type': 'application/json' },
      });
    }

    // 1. Verify Firebase ID token via Google Identity Toolkit REST API
    const FIREBASE_API_KEY = Deno.env.get('FIREBASE_API_KEY')!;
    const verifyResponse = await fetch(
      `https://identitytoolkit.googleapis.com/v1/accounts:lookup?key=${FIREBASE_API_KEY}`,
      {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ idToken: firebaseToken }),
      }
    );

    if (!verifyResponse.ok) {
      return new Response(JSON.stringify({ error: 'Invalid Firebase token' }), {
        status: 401,
        headers: { ...corsHeaders, 'Content-Type': 'application/json' },
      });
    }

    const verifyData = await verifyResponse.json();
    const firebaseUser = verifyData.users?.[0];
    if (!firebaseUser) {
      return new Response(JSON.stringify({ error: 'Firebase user not found' }), {
        status: 404,
        headers: { ...corsHeaders, 'Content-Type': 'application/json' },
      });
    }

    const firebaseUid = firebaseUser.localId;
    const phone = firebaseUser.phoneNumber;

    if (!phone) {
      return new Response(JSON.stringify({ error: 'No phone number on Firebase account' }), {
        status: 400,
        headers: { ...corsHeaders, 'Content-Type': 'application/json' },
      });
    }

    // 2. Create Supabase admin client with service_role key
    const supabaseUrl = Deno.env.get('SUPABASE_URL')!;
    const supabaseServiceKey = Deno.env.get('SUPABASE_SERVICE_ROLE_KEY')!;
    const supabaseAdmin = createClient(supabaseUrl, supabaseServiceKey);

    // 3. Check if user already migrated (lookup by firebase_uid)
    const { data: existingUser } = await supabaseAdmin
      .from('users')
      .select('id')
      .eq('firebase_uid', firebaseUid)
      .single();

    let supabaseUserId: string;

    if (existingUser) {
      // Already migrated -- use existing Supabase user ID
      supabaseUserId = existingUser.id;
    } else {
      // First migration -- create Supabase auth user with phone
      const { data: newAuthUser, error: createError } = await supabaseAdmin.auth.admin.createUser({
        phone,
        phone_confirm: true,
        user_metadata: { firebase_uid: firebaseUid },
      });

      if (createError) {
        // Phone may already exist in Supabase Auth (user created via OTP but not yet mapped)
        // Try to find by phone
        const { data: { users } } = await supabaseAdmin.auth.admin.listUsers();
        const existingAuthUser = users?.find((u) => u.phone === phone);
        if (existingAuthUser) {
          supabaseUserId = existingAuthUser.id;
        } else {
          return new Response(JSON.stringify({ error: 'Failed to create Supabase user' }), {
            status: 500,
            headers: { ...corsHeaders, 'Content-Type': 'application/json' },
          });
        }
      } else {
        supabaseUserId = newAuthUser.user.id;
      }

      // Update users table with firebase_uid mapping
      await supabaseAdmin
        .from('users')
        .update({ firebase_uid: firebaseUid })
        .eq('id', supabaseUserId);
    }

    // 4. Generate a magic link / OTP-less session for the user
    // Use generateLink to get a token the client can use
    const { data: linkData, error: linkError } = await supabaseAdmin.auth.admin.generateLink({
      type: 'magiclink',
      email: `${supabaseUserId}@phone.internal`, // Placeholder -- phone users don't have email
    });

    // Alternative: directly return user info and let client call signInWithOtp
    // If generateLink doesn't work for phone-only users, return user ID and trigger OTP
    if (linkError) {
      // Fallback: return the supabase user ID so client knows migration succeeded
      // Client will need to do a one-time OTP verification
      return new Response(JSON.stringify({
        success: true,
        migrated: true,
        requiresVerification: true,
        supabaseUserId,
        phone,
      }), {
        status: 200,
        headers: { ...corsHeaders, 'Content-Type': 'application/json' },
      });
    }

    return new Response(JSON.stringify({
      success: true,
      migrated: !existingUser,
      supabaseUserId,
    }), {
      status: 200,
      headers: { ...corsHeaders, 'Content-Type': 'application/json' },
    });

  } catch (error) {
    return new Response(JSON.stringify({ error: error.message }), {
      status: 500,
      headers: { ...corsHeaders, 'Content-Type': 'application/json' },
    });
  }
});
```
