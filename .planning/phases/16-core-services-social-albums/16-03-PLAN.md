Import `supabase` from `@/lib/supabase` and `logger` from `@/utils/logger`.

Define types:
```typescript
export interface Album {
  id: string;
  userId: string;
  title: string;
  type: 'custom' | 'monthly';
  monthKey: string | null;
  coverPhotoId: string | null;
  createdAt: string;
  photoCount?: number;
}

export interface MonthlyAlbumGroup {
  monthKey: string;
  photoCount: number;
  photos: Array<{
    id: string;
    imageUrl: string;
    createdAt: string;
    photoState: string;
  }>;
}
```

Constants: `MAX_TITLE_LENGTH = 24`.

1. `createAlbum(userId: string, title: string, photoIds: string[]): Promise<Album>` -- Validate: `title.trim()` must be 1-24 chars; `photoIds` must be non-empty (throw `new Error('Album name is required')` / `new Error('At least one photo is required')`). Insert into `albums`: `{ user_id: userId, title: title.trim(), type: 'custom', cover_photo_id: photoIds[0] }`. Then insert into `album_photos`: `photoIds.map(photoId => ({ album_id: albumId, photo_id: photoId }))`. Return mapped album.

2. `getAlbum(albumId: string): Promise<Album & { photos: string[] }>` -- Fetch album: `supabase.from('albums').select('*').eq('id', albumId).single()`. Fetch photos: `supabase.from('album_photos').select('photo_id').eq('album_id', albumId).order('added_at', { ascending: true })`. Combine and return.

3. `getUserAlbums(userId: string): Promise<Album[]>` -- `supabase.from('albums').select('*, album_photos(count)').eq('user_id', userId).eq('type', 'custom').order('created_at', { ascending: false })`. Map to include photoCount from the count aggregation.

4. `updateAlbum(albumId: string, updates: { title?: string; coverPhotoId?: string }): Promise<void>` -- Build update object: if title, validate length. Map to snake_case: `{ title: updates.title, cover_photo_id: updates.coverPhotoId }`. `supabase.from('albums').update(mapped).eq('id', albumId)`.

5. `deleteAlbum(albumId: string): Promise<void>` -- `supabase.from('albums').delete().eq('id', albumId)`. CASCADE removes album_photos.

6. `addPhotosToAlbum(albumId: string, photoIds: string[]): Promise<void>` -- `supabase.from('album_photos').insert(photoIds.map(photoId => ({ album_id: albumId, photo_id: photoId })))`.

7. `removePhotoFromAlbum(albumId: string, photoId: string): Promise<void>` -- `supabase.from('album_photos').delete().eq('album_id', albumId).eq('photo_id', photoId)`.

8. `setCoverPhoto(albumId: string, photoId: string): Promise<void>` -- `supabase.from('albums').update({ cover_photo_id: photoId }).eq('id', albumId)`.

9. `getMonthlyPhotos(userId: string): Promise<MonthlyAlbumGroup[]>` -- `supabase.rpc('get_monthly_photos', { target_user_id: userId })`. Map snake_case response to camelCase MonthlyAlbumGroup[].

**Create src/hooks/useAlbums.ts:**

Import from `@tanstack/react-query`, `@/lib/queryKeys`, `@/services/supabase/albumService`, `@/utils/logger`.

Export these hooks:

1. `useUserAlbums(userId: string)` -- `useQuery({ queryKey: queryKeys.albums.list(userId), queryFn: () => albumService.getUserAlbums(userId), enabled: !!userId })`.

2. `useAlbum(albumId: string)` -- `useQuery({ queryKey: queryKeys.albums.detail(albumId), queryFn: () => albumService.getAlbum(albumId), enabled: !!albumId })`.

3. `useCreateAlbum()` -- `useMutation({ mutationFn: ({userId, title, photoIds}) => albumService.createAlbum(userId, title, photoIds), onSuccess: (data) => queryClient.invalidateQueries({ queryKey: queryKeys.albums.list(data.userId) }) })`.

4. `useUpdateAlbum()` -- `useMutation({ mutationFn: ({albumId, updates}) => albumService.updateAlbum(albumId, updates), onSettled: (_, __, {albumId}) => queryClient.invalidateQueries({ queryKey: queryKeys.albums.detail(albumId) }) })`.

5. `useDeleteAlbum()` -- `useMutation({ mutationFn: ({albumId, userId}) => albumService.deleteAlbum(albumId), onSuccess: (_, {userId}) => queryClient.invalidateQueries({ queryKey: queryKeys.albums.list(userId) }) })`.

6. `useAddPhotosToAlbum()` -- TanStack useMutation with **optimistic update**:
   - onMutate: cancel queries for album detail, get previous data, optimistically add photo IDs to cached album
   - onError: rollback to previous
   - onSettled: invalidate album detail query

7. `useRemovePhotoFromAlbum()` -- TanStack useMutation with **optimistic update**:
   - onMutate: optimistically remove photo from cached album photos list
   - onError: rollback
   - onSettled: invalidate

8. `useSetCoverPhoto()` -- `useMutation({ mutationFn: ({albumId, photoId}) => albumService.setCoverPhoto(albumId, photoId), onSettled: (_, __, {albumId}) => queryClient.invalidateQueries({ queryKey: queryKeys.albums.detail(albumId) }) })`.
