  <action>
    Create `src/utils/imageUrl.ts` with the following exports:

    1. `FEED_CARD_WIDTH = 400` constant

    2. `getTransformedPhotoUrl(storagePath: string, options?: { width?: number; quality?: number; bucket?: string }): string`
       - Use when you have the storage path (e.g., from new uploads or when storagePath is available)
       - Default bucket: 'photos'
       - If no width provided: call `supabase.storage.from(bucket).getPublicUrl(storagePath)` and return `data.publicUrl`
       - If width provided: call `supabase.storage.from(bucket).getPublicUrl(storagePath, { transform: { width, ...(quality ? { quality } : {}) } })` and return `data.publicUrl`

    3. **`appendTransformParams(fullUrl: string, options?: { width?: number; quality?: number }): string`** (NEW -- addresses review HIGH concern about storagePath vs fullUrl mismatch)
       - Use when you have a FULL public CDN URL (e.g., FeedPhoto.imageUrl from feed RPC)
       - If no width provided, return fullUrl unchanged
       - If URL contains '/storage/v1/object/public/', replace with '/storage/v1/render/image/public/' and append `?width={width}&format=webp`
       - If quality provided, also append `&quality={quality}`
       - If URL does NOT contain '/storage/v1/object/public/' (e.g., external URL), return fullUrl unchanged (graceful no-op)
       - This avoids needing to parse the storage path from a full URL

    4. `getSignedUrlExpiry(signedUrl: string): number | null`
       - Parse the URL, extract `token` query parameter
       - If no token, return null
       - Decode JWT payload (second segment): **First try `Buffer.from(token.split('.')[1], 'base64').toString()`**. If Buffer is not available (catch ReferenceError), fall back to a manual base64 decode using `decodeURIComponent(escape(atob(...)))`. This addresses the review MEDIUM concern about Buffer.from() availability.
       - Import: `let BufferImpl: typeof Buffer | undefined; try { BufferImpl = require('buffer').Buffer; } catch { /* Buffer not available */ }`
       - Parse JSON payload, return `payload.exp * 1000` (convert seconds to ms)
       - Wrap in try/catch, return null on any error

    5. `isUrlNearExpiry(signedUrl: string, thresholdMs = 60_000): boolean`
       - Call getSignedUrlExpiry, if null return true (can't determine = treat as expired, fail-safe)
       - Return `Date.now() + thresholdMs >= expiry`
