  <action>
Create `supabase/functions/send-live-activity/index.ts` -- Edge Function for APNS HTTP/2 Live Activity push delivery. Port logic from `functions/notifications/liveActivitySender.js`.

The function:

1. **Validate Authorization header** (service_role key)
2. **Parse JSON body:** `{ user_id: string, action: 'start' | 'update' | 'end', content_state: object, stale_date?: number }`
3. **Look up user's push_to_start_token** from users table via Supabase service_role client
4. **Generate APNS JWT** using ES256 signing:
   - Header: `{ alg: 'ES256', kid: APNS_KEY_ID }`
   - Claims: `{ iss: APNS_TEAM_ID, iat: nowSeconds }`
   - Sign with `APNS_AUTH_KEY_P8` using `node:crypto` createSign('SHA256')
   - Cache JWT for 50 minutes (3000 seconds) -- APNS JWTs valid for 1 hour
   - Normalize PEM key: strip BEGIN/END markers and whitespace, reconstruct with proper line breaks
5. **Build APNS payload:**
   ```json
   {
     "aps": {
       "timestamp": nowSeconds,
       "event": action,       // "start", "update", "end"
       "content-state": content_state,
       "stale-date": stale_date,
       "relevance-score": 100,
       "alert": {
         "title": content_state.senderName || "Flick",
         "body": "pinned a snap to your screen"
       }
     }
   }
   ```
6. **Send via HTTP/2** using `node:http2` connect():
   - Primary: `https://api.push.apple.com` (production)
   - Headers: `:method` POST, `:path` `/3/device/{token}`, `authorization` bearer JWT, `apns-topic` `com.spoodsjs.flick.push-type.liveactivity`, `apns-push-type` `liveactivity`, `apns-priority` `10`
   - If primary returns 400 with reason `BadDeviceToken`, retry with sandbox: `https://api.sandbox.push.apple.com`
   - This dual-environment fallback matches the existing liveActivitySender.js behavior
7. **Return JSON response:**
   - Success: `{ success: true, environment: 'production' | 'sandbox' }`
   - Token missing: `{ success: false, error: 'No push_to_start_token' }`
   - APNS error: `{ success: false, error: reason, statusCode: N }`

**Environment variables** (set via `supabase secrets set`):
- `APNS_KEY_ID` -- Apple APNs key ID
- `APNS_TEAM_ID` -- Apple Developer Team ID
- `APNS_AUTH_KEY_P8` -- Contents of the .p8 private key file

**Bundle ID:** `com.spoodsjs.flick` (from CLAUDE.md)
**APNS topic for Live Activity:** `com.spoodsjs.flick.push-type.liveactivity`

Handle CORS and OPTIONS preflight.
Use `import { connect } from 'node:http2'` and `import { createSign } from 'node:crypto'` (Deno built-in Node compat).
Use `import { createClient } from 'npm:@supabase/supabase-js@2'` for Supabase.
