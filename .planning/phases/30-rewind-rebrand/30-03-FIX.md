<task type="checkpoint:human-action" gate="blocking">
  <action>Update Google Cloud Console project display name</action>
  <instructions>
Google Cloud project names cannot be changed after creation, but display names can be edited:

1. Go to Google Cloud Console: https://console.cloud.google.com/
2. Select your project from the dropdown
3. Click "Navigation menu" (hamburger) → "IAM & Admin" → "Settings"
4. Click "Edit" next to the project name
5. Update "Project name" to "Rewind" (this is the display name, not the project ID)
6. Click Save

**OAuth Client Display Names:** 7. Navigate to "APIs & Services" → "Credentials" 8. Click on each OAuth 2.0 Client ID:

- iOS Client: Update name to "Rewind iOS Client"
- Web Client: Update name to "Rewind Web Client" (if exists)
- Android Client: Update name to "Rewind Android Client" (if exists)

9. Save each change
   </instructions>
   <verification>Google Cloud Console shows "Rewind" as project display name and OAuth clients are renamed</verification>
   <resume-signal>Type "done" when Google Cloud Console is updated</resume-signal>
   </task>
