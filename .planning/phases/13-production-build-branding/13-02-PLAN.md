**Structure:**

const AnimatedSplash = ({ onAnimationComplete }) => {
  const openProgress = useSharedValue(0);

<task type="auto">
  <name>Task 3: Integrate animated splash into App.js</name>
  <files>App.js</files>
  <action>
Update App.js to show AnimatedSplash on launch:

**Implementation:**
1. Import SplashScreen from expo-splash-screen
2. Import AnimatedSplash component
3. Add state: showSplash = true
4. Call SplashScreen.preventAutoHideAsync() before component mounts (top-level)
5. Render AnimatedSplash when showSplash is true
6. In onAnimationComplete callback:
   - Call SplashScreen.hideAsync()
   - Set showSplash = false

**Code pattern:**
```javascript
import * as SplashScreen from 'expo-splash-screen';
import { AnimatedSplash } from './src/components';

// Prevent auto-hide before component mounts
SplashScreen.preventAutoHideAsync();

export default function App() {
  const [showSplash, setShowSplash] = useState(true);

  const handleSplashComplete = async () => {
    await SplashScreen.hideAsync();
    setShowSplash(false);
  };

  return (
    <>
      {/* Existing app content */}
      {showSplash && <AnimatedSplash onAnimationComplete={handleSplashComplete} />}
    </>
  );
}
```

The AnimatedSplash should render OVER the app content (absolute positioning) so the app is ready behind it.
