# Lapse Clone

A friends-only social media app inspired by disposable cameras. Built with React Native and Expo.

Photos are captured instantly but revealed later in batches through a "darkroom" system, encouraging authentic moments over polished content. Features include friend requests, emoji reactions, and push notifications.

## Tech Stack

- React Native + Expo
- Firebase (Auth, Firestore, Cloud Storage, Cloud Functions)
- Expo Push Notifications
